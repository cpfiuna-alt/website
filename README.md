
# Club de Programación FIUNA - Sitio Web Oficial

Este repositorio contiene el código fuente del sitio web oficial del Club de Programación de la Facultad de Ingeniería de la Universidad Nacional de Asunción (FIUNA).

## Sobre el Proyecto

El Club de Programación FIUNA es una comunidad de estudiantes y profesionales apasionados por la programación y el desarrollo de software en la Facultad de Ingeniería de la Universidad Nacional de Asunción. Este sitio web sirve como plataforma central para nuestras actividades, eventos, proyectos y recursos.

## Estructura del Sitio

El sitio web está organizado en varias secciones principales:

- **Inicio**: Página principal con información general sobre el club y actividades destacadas.
- **Sobre Nosotros**: Historia, misión, visión y valores del club, junto con información sobre la Comisión Directiva y Secretarías.
- **Eventos**: Calendario de eventos, talleres, competencias y actividades organizadas por el club.
- **Proyectos**: Proyectos desarrollados por miembros del club.
- **Recursos**: Material educativo, guías de aprendizaje, roadmaps, cursos y recursos externos.
- **Blog**: Artículos técnicos, tutoriales y noticias relacionadas con la tecnología.
- **Contacto**: Información de contacto y formulario para comunicarse con el club.
- **Legal**: Estatuto del club, políticas de privacidad, código de conducta y transparencia.
- **Prensa**: Apariciones en medios y material para prensa.
- **Kit de Medios**: Recursos gráficos y multimedia para uso de la prensa y colaboradores.

## Tecnologías Utilizadas

El sitio está construido con tecnologías modernas de desarrollo web:

- **React**: Biblioteca para construir interfaces de usuario
- **TypeScript**: Superset de JavaScript tipado
- **Vite**: Herramienta de compilación ultrarrápida
- **Tailwind CSS**: Framework CSS utilitario para diseño responsivo
- **Shadcn/UI**: Componentes de interfaz de usuario reutilizables
- **React Router**: Enrutamiento para aplicaciones React
- **Framer Motion**: Biblioteca para animaciones
- **React Query**: Manejo del estado de datos del servidor
- **Lucide**: Biblioteca de iconos SVG
- **Recharts**: Biblioteca para visualización de datos

## Características Principales

- Diseño responsive para todos los dispositivos
- Modo claro/oscuro basado en las preferencias del usuario
- Animaciones y transiciones fluidas
- Navegación intuitiva
- Visualización completa de eventos y proyectos
- Recursos educativos organizados
- Secciones de transparencia y documentación legal
